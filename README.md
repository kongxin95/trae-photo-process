# PixelCraft · 极简纯前端图片处理工具

> 纯前端运行 · 零后端 · 零上传 · 永久免费
>
> 所有图片裁剪、压缩与格式转换都在你的浏览器本地完成，图片不离开设备，隐私由你掌控。

PixelCraft 是一个单文件、零依赖、打开即用的在线图片处理工具。无需安装、无需服务器，双击 HTML 即可运行，适合内容创作者、设计师、自媒体运营者及对隐私敏感的用户在日常工作中快速完成图片裁剪、压缩与格式转换。

## 🚀 快速开始

### 方式一：直接打开（最简单）
双击 `index.html`，用任意现代浏览器打开即可使用。

### 方式二：本地静态服务器（推荐，避免部分浏览器对本地文件的限制）
在项目目录下任选一种方式启动：

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server -p 8000

# PHP
php -S 127.0.0.1:8000
```

然后访问 `http://127.0.0.1:8000/index.html`。

> 推荐使用 Chrome / Edge / Firefox 最新版本，以获得最佳 WebP 编码与 `createImageBitmap` 性能。

## 🛠 技术栈

- **HTML5** — 语义化结构
- **Tailwind CSS**（Play CDN） — 原子化样式 + 自定义主题
- **原生 JavaScript（ES6+）** — 全部业务与处理逻辑，无框架依赖
- **Canvas 2D API** — 图片裁剪、重绘与编码（`drawImage` + `toBlob`）
- **中值切分颜色量化** — PNG 8 位索引色压缩（median cut + deflate）
- **JSZip 3.x**（CDN） — 浏览器端 ZIP 打包
- **Google Fonts** — Sora / Manrope / JetBrains Mono

无构建步骤、无包管理、无后端、无数据库。

## 🧭 使用流程

1. 打开 `index.html`（默认亮色主题，可右上角切换）。
2. 把图片拖到上传区，或点击上传区选择文件（可多选）。
3. 在左侧「处理设置」面板选择输出格式（JPG / PNG / WebP）。
4. 拖动质量滑动条，右侧卡片与顶部「预估体积」面板会实时更新。
5. **裁剪图片**：点击卡片左侧裁剪按钮，拖动裁剪框调整区域，可选择比例预设，确认后自动重新处理。
6. **对比效果**：点击卡片放大查看，左右拖动对比条直观对比原图与处理后差异。
7. 单张点击卡片内「下载」按钮导出；或使用页面底部吸底工具条：
   - **打包 ZIP**：把所有已处理图片压缩为一个 `.zip` 下载。
   - **逐张下载**：按顺序逐张触发浏览器下载。
8. 处理完成后可点击「清空全部」重置工作台。

## ⚙️ 高级与自定义

### 接入广告
页面预留两处广告位占位（带 `AD · 广告位` 水印的虚线卡片）：
- 顶部 banner 位（728 × 90）
- 赞赏区侧栏位（300 × 250）

将对应占位 `<div class="ad-slot ...">...</div>` 替换为你的联盟广告代码（`<script>` / `<iframe>` / `<ins>`）即可，无需改动 JS 逻辑。

### 替换赞赏码
页面底部「请作者喝杯咖啡」区域有两个二维码框（微信 + 支付宝）：

- 将 `png/wechat.png` 替换为你的微信赞赏码图片
- 将 `png/alipay.png` 替换为你的支付宝收款码图片

文件名保持不变即可自动加载，或修改 HTML 中的 `src` 属性。

### 调整强调色
在 `index.html` 顶部的 `<style>` 块中修改 CSS 变量即可全局换色：

```css
:root {
  --accent: #7CFFB2;   /* 主强调色，电光青（暗色模式） */
  --grape: #A78BFA;    /* 辅助色，冷紫（暗色模式） */
}
html.light {
  --accent: #0EA57E;   /* 主强调色，深翠绿（亮色模式） */
  --grape: #7C3AED;    /* 辅助色，深紫（亮色模式） */
}
```

为什么选择 PixelCraft

专为隐私与效率而生的极简图片工具。

隐私优先

图片全程在本地处理，永不上传服务器，从根上杜绝泄露。

极速响应

Canvas API 本地编码，滑动质量即刻预览体积，无需等待网络。

批量打包

多图同传，一键 ZIP 打包或逐张下载，工作流丝滑顺畅。

EXIF 隐私

一键抹除 GPS 定位、拍摄时间、相机型号等隐私元数据，安全分享。

格式自由

JPG / PNG / WebP 互转，裁剪尺寸，压缩质量从 0.1 到 1.0 精细可控。

平台适配

小红书、公众号、手机壁纸等主流平台尺寸，一键裁剪无需手动调整。
